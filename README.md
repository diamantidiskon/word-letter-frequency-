Οδηγίες χρήσης:
• Εκτέλεσε το πρόγραμμα σε Python 3.
• Όταν ζητηθεί, δώσε το όνομα του αρχείου .txt που θέλεις να αναλύσεις (π.χ. input.txt).

Το πρόγραμμα θα:
1) διαβάσει το περιεχόμενο του αρχείου.
2) υπολογίσει συχνότητες γραμμάτων και λέξεων.
3) εμφανίσει τους δέκα πιο συχνούς χαρακτήρες και λέξεις στην οθόνη.
4) δημιουργήσει τα αρχεία characters.txt και words.txt με αναλυτικά δεδομένα.

Σημειώσεις:
• Το πρόγραμμα αναλύει μόνο αγγλικά γράμματα (a–z).

Απαιτήσεις:
• Περιβάλλον Python έκδοσης 3.x
• Αρχείο κειμένου .txt σε κωδικοποίηση UTF-8
